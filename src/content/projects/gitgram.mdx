## Background

Gitgram is a bot that can receive Github webhook events and forward it to Telegram.

I made this because at that time our team was working on a big project and we use Telegram to discuss the project. We needed something to receive Github webhook notifications and send them to the Telegram group to make our life easier. It's pretty simple, but it works.
